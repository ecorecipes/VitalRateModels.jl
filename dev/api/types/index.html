Fitted model containers

VitalRateModels.FittedSurvival — Type
FittedSurvival

A fitted survival model. Predicts P(survival | size, covariates).

Fields

  • model: The underlying fitted GLM/GAM object
  • formula: The model formula
  • distribution: Distribution family used
  • aic: AIC of the fitted model
  • n_obs: Number of observations
source
VitalRateModels.FittedGrowth — Type
FittedGrowth

A fitted growth model. Predicts size(t+1) | size(t), covariates.

Fields

  • model: Fitted model for mean response
  • sigma: Residual standard deviation (for Gaussian)
  • formula: Model formula
  • distribution: Distribution family
  • aic: AIC
  • n_obs: Number of observations
source

A fitted fecundity model. Predicts reproductive output | size, covariates.

source
n = 120
size_t = clamp.(rand(Normal(5.0, 1.0), n), 0.5, 9.0)
surv_prob = @. 1 / (1 + exp(-(-1.8 + 0.45 * size_t)))
survived = rand.(Bernoulli.(surv_prob)) .== 1
size_t1 = clamp.(0.9 .+ 0.95 .* size_t .+ rand(Normal(0, 0.5), n), 0.0, 10.5)
fecundity = rand.(Poisson.(exp.(-2.5 .+ 0.25 .* size_t)))
recruit_size = clamp.(rand.(Normal.(1.0 .+ 0.03 .* size_t, 0.25)), 0.1, 2.5)

demo = DataFrame(
    size_t=size_t,
    size_t1=size_t1,
    survived=survived,
    fecundity=fecundity,
    recruit_size=recruit_size,
)

survival_fit = fit_vital_rate(SurvivalModel, demo, @formula(survived ~ size_t))
growth_fit = fit_vital_rate(GrowthModel, demo, @formula(size_t1 ~ size_t))
fecundity_fit = fit_vital_rate(FecundityModel, demo, @formula(fecundity ~ size_t), distribution=Poisson_())
recruitment_fit = fit_vital_rate(RecruitmentModel, demo, @formula(recruit_size ~ size_t))

(typeof(survival_fit), typeof(growth_fit), typeof(fecundity_fit), typeof(recruitment_fit))
(FittedSurvival{StatsModels.TableRegressionModel{GLM.GeneralizedLinearModel{GLM.GlmResp{Vector{Float64}, Distributions.Bernoulli{Float64}, GLM.LogitLink}, GLM.DensePredChol{Float64, LinearAlgebra.CholeskyPivoted{Float64, Matrix{Float64}, Vector{Int64}}}}, Matrix{Float64}}}, FittedGrowth{StatsModels.TableRegressionModel{GLM.GeneralizedLinearModel{GLM.GlmResp{Vector{Float64}, Distributions.Normal{Float64}, GLM.IdentityLink}, GLM.DensePredChol{Float64, LinearAlgebra.CholeskyPivoted{Float64, Matrix{Float64}, Vector{Int64}}}}, Matrix{Float64}}}, FittedFecundity{StatsModels.TableRegressionModel{GLM.GeneralizedLinearModel{GLM.GlmResp{Vector{Float64}, Distributions.Poisson{Float64}, GLM.LogLink}, GLM.DensePredChol{Float64, LinearAlgebra.CholeskyPivoted{Float64, Matrix{Float64}, Vector{Int64}}}}, Matrix{Float64}}, Nothing}, FittedRecruitment{StatsModels.TableRegressionModel{GLM.GeneralizedLinearModel{GLM.GlmResp{Vector{Float64}, Distributions.Normal{Float64}, GLM.IdentityLink}, GLM.DensePredChol{Float64, LinearAlgebra.CholeskyPivoted{Float64, Matrix{Float64}, Vector{Int64}}}}, Matrix{Float64}}})

Stage frames

VitalRateModels.StageFrame — Type
StageFrame

A stage classification framework for structured population models.

Fields

  • stage_names: Names of life stages
  • sizes: Representative size for each stage (midpoint or mean)
  • bin_halfwidths: Half-width of size bins for IPM-style classification
  • reproductive: Whether each stage is reproductive
  • observable: Whether each stage is observable in the field
  • mature: Whether each stage is mature
  • propagule: Whether each stage is a propagule/seed stage
  • dormant: Whether each stage is dormant/unobservable
source
stageframe = create_stageframe(
    stage_names=[:seedling, :small, :large, :flowering],
    sizes=[0.5, 2.5, 6.0, 9.0],
    reproductive=[false, false, false, true],
    observable=[true, true, true, true],
)
